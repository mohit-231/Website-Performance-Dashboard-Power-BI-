# Website Performance Dashboard using Power BI

## Project Overview
This project focuses on analyzing website performance metrics using Power BI. The dashboard provides interactive visualizations and analytical insights related to website traffic, user engagement, and performance trends to support data-driven business decisions.
The project demonstrates practical applications of business intelligence, data visualization, KPI tracking, and dashboard development.
